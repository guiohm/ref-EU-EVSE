Explications

      
    
        
  • La table logs accessible ici est une jointure avec les données du fichier source sur le station_id et pdc_id (si présent)
    • 
        
  • La colonne log_id correspond à une unique entrée de log. Une ligne de log apparait autant de fois qu'il y a d'entrées dans le fichier source correspondant (jointure)
    • 
        
  • Les colonnes du fichier de log ont un fond rouge : log_id, level, station_id, source, msg, detail, pdc_id
    • 
      

      
Fonctionnalités

      
    
        
  • Un clic dans la liste des colonnes colle automatiquement celle-ci dans l'éditeur
    • 
        
  • Un clic dans le tableau de résultats sélectionne la cellule. Ctrl+V colle cette sélection sans effacer le presse-papier
    • 
        
  • L'URL du navigateur contient la dernière requête exécutée
    • 
        
  • Chaque requête est donc enregistrée dans l'historique du navigateur pour une navigation plus naturelle
    •
